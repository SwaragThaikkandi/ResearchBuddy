        graph database backend

      
Persistent graph storage + the Neo4j Browser for
        interactive Cypher exploration. ResearchBuddy works fine on the
        built-in NetworkX backend; Neo4j adds durability and live queries.

      

      

        
        
        
        
          
        
      

    

    

      
 GROBID
        scientific-PDF parser

      
Dramatically better PDF extraction (sections,
        references, figures, equations). Without it, imports fall back to
        pdfplumber — usable, but shallower graph nodes. Start it BEFORE
        importing PDFs in the Library tab.

      

        
        
        
      

    

    

      
 Local LLM (Ollama)
        HyDE · query expansion · reranking · synthesis

      

      
Runs outside Docker: ollama serve then
        ollama pull phi3.5. Search works without it — quality is
        just reduced.
